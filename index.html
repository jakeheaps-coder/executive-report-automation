Weekly Executive Report Automation

An AI-powered pipeline that collects data from 3 sources, generates a rubric-graded executive report with Gemini, and delivers branded Word + PowerPoint documents to leadership every week.

01 — Pipeline

7 Steps, Fully Automated

Every Friday at 9am, Cloud Scheduler triggers a Cloud Run job that executes the full pipeline end-to-end.

02 — Data Sources

Three Sources of Truth

The pipeline collects structured, unstructured, and visual data from three primary sources with strict priority ordering.

Domo Dashboard

38 cards exported as PDF for Gemini Vision analysis, plus notebook card text extraction and individual table renders.

Primary #1
  • Full page PDF export (scale 2.0)
  • Notebook text cards via API
  • Individual table card renders
  • Card metadata (38 cards)

MLT Report

Marketing Leadership Team Word document from SharePoint with initiative status updates across 7 functional areas.

Primary #2
  • Product & Integration updates
  • Paid Search, SEO/GEO status
  • AI & Outbound initiatives
  • Events, Comms & Brand, Regional

Marketing Calendar

Excel spreadsheet from SharePoint parsed with openpyxl. Contains event dates, registration counts, and activity timelines.

Secondary
  • All worksheets parsed to dicts
  • Event dates & registrations
  • Activity timeline data
  • Calendar section input only
03 — AI Generation

Gemini-Powered Synthesis

The pipeline uses Gemini for two distinct tasks: visual analysis of the dashboard PDF, and multi-round report generation with a structured 11-section prompt.

Step 1: Vision Analysis

Gemini analyzes the full dashboard PDF screenshot to extract every visible number, metric, trend, and regional breakdown.

  • Files >500KB use Gemini File API (resumable upload)
  • Smaller files sent inline as base64
  • Individual table cards rendered separately
  • Temperature: 0.2 (precision mode)

Step 2: Report Generation

Gemini synthesizes all source data into a structured executive report using an 11-section system prompt.

  • Model: gemini-3.1-pro-preview
  • Temperature: 0.3 (consistent, accurate)
  • Max tokens: 16,384
  • Up to 5 refinement rounds

11-Section System Prompt

The system instruction defines context, objectives, tone, data priority, output structure, and validation requirements.

04 — Quality Grading

Rubric-Graded, 5-Round Refinement

Every generated report is graded against a 5-criteria rubric. If it doesn't score 100, Gemini refines it with specific feedback.

05 — Deliverables

Three Outputs, Auto-Delivered

The pipeline produces a Word document, a branded PowerPoint slide, and sends email notifications with links.

Word Document (.docx)

Full executive report with headings, bullets, and formatted sections. Uploaded to SharePoint Executive Staff Prep folder.

PowerPoint Slide (.pptx)

Branded single-slide deck with pipeline drivers, key initiatives, and decisions/asks. Domo blue styling with logo.

Email Notification

Sent via Domo Code Engine with direct SharePoint links to both documents. Recipients: leadership team.

What the Report Contains

Every report follows a strict structure optimized for time-constrained CMO/executive consumption.

07 — Infrastructure

Cloud-Native, Production-Grade

Deployed on Google Cloud with automated scheduling, secret management, structured logging, and retry resilience.

Cloud Run Jobs

Serverless container execution

  • 2Gi memory, 2 CPUs
  • 15-minute timeout
  • Multi-stage Dockerfile (Python 3.11-slim)
  • Max 1 retry on failure

Cloud Scheduler

Automated weekly trigger

  • Cron: 0 9 * * 5 (Fridays 9am)
  • Timezone: America/Denver (MST)
  • OAuth2 service account auth
  • HTTP POST to Cloud Run Job

Reliability

Error handling and observability

  • Exponential backoff retries (3x)
  • Structured JSON logging (Cloud Logging)
  • Custom exception hierarchy
  • RUN_ID tracing per execution
  • Secrets via Google Secret Manager
